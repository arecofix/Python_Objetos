## Especificaciones Técnicas
- **Formato**: JPG para fotos, SVG para gráficos vectoriales
- **Tamaño recomendado**: 1920x1080px para el carousel
- **Calidad**: Alta calidad pero optimizada para web (máximo 500KB por imagen)
- **Proporción**: 16:9 para el carousel, 1:1 para imágenes cuadradas

## Notas
- Las imágenes deben mostrar trabajos reales y de alta calidad
- Es recomendable tener consentimiento de los clientes para mostrar sus propiedades
- Las imágenes deben estar bien iluminadas y mostrar claramente los resultados
