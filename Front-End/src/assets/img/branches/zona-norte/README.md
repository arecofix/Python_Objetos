## 📋 Especificaciones Técnicas

### Dimensiones Recomendadas
- **Carousel Hero**: 1920x1080px (16:9)
- **Servicios**: 600x400px (3:2)
- **Proyectos**: 400x300px (4:3)
- **Logos**: 200x200px (cuadrado)
- **Favicon**: 32x32px o 64x64px

### Optimización
- **Formato**: JPG para fotos, PNG para logos
- **Calidad**: Alta pero optimizada para web
- **Peso máximo**: 500KB por imagen
- **Compresión**: 80-90% de calidad

### Contenido Visual
- **Fotos reales** de trabajos realizados
- **Buena iluminación** y enfoque claro
- **Sin marcas de agua** de otros proveedores
- **Consentimiento** de clientes para mostrar propiedades

## 🔄 Actualización de Imágenes

### Para agregar nuevas imágenes:
1. Colocar las imágenes en la carpeta correspondiente
2. Usar nombres de archivo exactos como se especifican
3. Optimizar para web antes de subir
4. Probar que se carguen correctamente

### Para actualizar el carousel:
1. Reemplazar las imágenes `hero-1.jpg` a `hero-4.jpg`
2. Mantener la proporción 16:9
3. Asegurar buena calidad visual

## 📝 Notas Importantes

- Estas imágenes son **exclusivas** para la sucursal Zona Norte
- No comparten branding con Arecofix principal
- Se cargan automáticamente en las secciones correspondientes
- Si una imagen no existe, se muestra un placeholder
- Las imágenes deben ser profesionales y de alta calidad
