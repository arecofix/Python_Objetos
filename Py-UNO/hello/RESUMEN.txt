💻 EJEMPLO: CREAR UN ESTUDIANTE
═══════════════════════════════════════════════════════════════════════════

# PowerShell
$body = @{
    nombre = "Juan Pérez"
    email = "juan@example.com"
    carrera = "Ingeniería Informática"
} | ConvertTo-Json

Invoke-WebRequest `
  -Uri "http://127.0.0.1:5000/api/estudiantes" `
  -Method POST `
  -Headers @{"Content-Type"="application/json"} `
  -Body $body

📦 PRÓXIMOS PASOS (ESCALABILIDAD)
═══════════════════════════════════════════════════════════════════════════

Para convertir a aplicación de producción:

1. Base de Datos Real
   $ pip install Flask-SQLAlchemy postgresql

2. Validación de Datos
   $ pip install Marshmallow flask-marshmallow

3. Autenticación (JWT)
   $ pip install flask-jwt-extended

4. Testing Unitario
   $ pip install pytest pytest-flask

5. Documentación API
   $ pip install flasgger

6. Logging Avanzado
   - Integrar Python logging
   - Usar Sentry para monitoreo

7. Containerización
   - Crear Dockerfile
   - docker-compose.yml

8. CI/CD
   - GitHub Actions
   - Deploy a Heroku/AWS/GCP
